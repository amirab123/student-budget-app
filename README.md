
Gestion de Budget
Description du projet
Dans le cadre de ce projet, vous serez amenés à concevoir et développer une application web de gestion de budget orientée vers les étudiants.

L’objectif est de créer un outil interactif et intuitif permettant aux utilisateurs de suivre leurs revenus et leurs dépenses de manière efficace.

Ce projet permet de consolider les compétences en développement d’interfaces web et en conception d’expériences utilisateur tout en appliquant les bonnes pratiques de développement.

Les technologies utilisées peuvent inclure :

ReactJS
Tailwind CSS
Toute autre technologie pertinente selon les besoins du projet
Structure du projet
L’application comprend les pages suivantes :

Landing Page

Première page visible par les utilisateurs.
Contenu suggéré : Lorem Ipsum.
Design au choix.
Page d’inscription

Permet aux utilisateurs de créer un compte.
Page de connexion

Permet aux utilisateurs d’accéder à leur espace personnel.
Page de profil

Pour éditer et mettre à jour le profil de l’utilisateur.
Page de budget mensuel

Affiche un résumé des revenus et dépenses du mois en cours.
Fournit des outils de suivi et d’analyse.
